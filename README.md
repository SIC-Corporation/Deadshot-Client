# 🔴 NexaFlow Client for DeadShot
> **The Official High-Performance Desktop Client by SIC Corporation**

NexaFlow is a premium, lightweight Electron-based client specifically engineered for the 3D FPS browser game `deadshot.io`. It is optimized for maximum frame rates and a seamless competitive gaming experience.

---

## 📥 Download & Install
**The fastest way to get in the game with NexaFlow performance.**

1. Go to the [**Latest Releases**](https://github.com/SIC-Corporation/Deadshot-Client/releases) page.
2. Download the `NexaFlow-Client-Setup.exe`.
3. Run the installer and jump into the action!

---

## 🚀 Key Features
* **Uncapped FPS:** Experience the game without artificial limits for the smoothest gameplay possible.
* **NexaHUD:** A custom, high-refresh integrated FPS counter built directly into the UI.
* **Discord Rich Presence:** Show your status, map, and score automatically on Discord.
* **Smooth Play & Anti-Lag:** Built-in contrast enhancements and performance optimizations to reduce input lag.
* **SIC Corp Compatibility:** Fully integrated with SIC Corp standards and NexaFlow services.
* **Custom UI:** Specialized red-and-black themed menus, custom scrollbars, and sleek overlays.
